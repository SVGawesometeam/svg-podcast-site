About the Guest

Thomas Dohmke
CEO, GitHub

Thomas Dohmke is the CEO of GitHub, the world's largest software development platform with over 100 million users. He led the company's $7.5 billion integration with Microsoft and oversaw the launch of GitHub Copilot, which became the most widely adopted AI coding tool in history. Dohmke is a leading voice on the future of software development, AI agents, and developer education.

In this episode of the Silicon Valley Girl Podcast, Marina Mogilko interviews Thomas Dohmke, CEO, GitHub. Marina Mogilko interviews GitHub CEO Thomas Dohmke at VivaTech in Paris about the future of software development in the age of AI. Dohmke explains how vibe coding works, why coding skills remain essential even as AI handles more of the work, and why he believes there will be more developers in five years, not fewer. They also discuss AGI timelines, skills kids should learn, and the top AI tools developers should be using today.

Key Takeaways

  • Vibe coding lets anyone interact with AI agents to build software without reviewing every line of code, but complexity hits a ceiling fast — scaling a site for traffic like a Black Friday rush still requires a professional developer.
  • Having only AI-assisted, no-code skills will not differentiate a business, because if anyone can prompt the same product in five minutes, there is no competitive advantage and no path to a billion-dollar company.
  • Dohmke predicts far more developers in five years, not fewer, because AI dramatically lowers the barrier to entry and makes learning to code more accessible, especially for kids.
  • The smartest companies will hire more developers, not fewer, using AI to build 10x to 100x more complex products and widen the gap between themselves and competitors who rely solely on no-code tools.
  • AI agents are expected to write roughly 90% of code in the near future, but human developers will remain essential for directing, validating, and scaling what those agents produce.
